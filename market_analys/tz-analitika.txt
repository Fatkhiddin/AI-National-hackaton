## 📊 MA'LUMOTLAR MANBASI

### Google Sheets (3 ta fayl)
- **URL 1**: https://docs.google.com/spreadsheets/d/1hVwC09Wlz4HPcQCnZpkGsznFHEfZ0Z9B575Sgq8aaRk/edit?usp=sharing
- **URL 2**: https://docs.google.com/spreadsheets/d/1OCjZjtwIV4rzKRDoGsudOus3rSIzN6zzT8nFsysLcRw/edit?usp=drivesdk
- **URL 3**: https://docs.google.com/spreadsheets/d/11hg22AxGAv2yKkZbRbIUreXnjGASyrvIIZIkk-eVMUk/edit?usp=drivesdk

### Har bir fayl ichida 2 ta sheet:
1. **Remontli** - ta'mirlangan uylar uchun narxlar
2. **Remontsiz** - ta'mirlanmagan uylar uchun narxlar

### Ma'lumotlar Strukturasi (CSV format):
```
T/r | Etaj | Xonalar soni | Qurilish turi | Maydon | Arzon | Bozor | Qimmat
1   | 1    | 2           | Gʻishtli      | 45     | 25000 | 28000 | 31000
```

**Ustunlar:**
- `T/r` - Tartib raqami (ignore)
- `Etaj` - Qavat raqami (integer)
- `Xonalar soni` - Xonalar soni (integer)
- `Qurilish turi` - Bino turi: "Gʻishtli", "Panelli", "Monolitli", "Blokli"
- `Maydon` - maksimal maydon (integer, m²)
- `Arzon` - Minimal narx (decimal, so'm/m²)
- `Bozor` - O'rtacha bozor narxi (decimal, so'm/m²)
- `Qimmat` - Maksimal narx (decimal, so'm/m²)

**MUHIM QOIDALAR:**
- Agar narx = 0 yoki bo'sh, bu qatorni ignore qilish
- Maydon diapazonini hisoblash: maydon to maydon+5 (masalan: 45-50m²)

## 🗄️ DATABASE MODELS

### Model 1: MarketPriceReference

**Maqsad:** Google Sheets'dan import qilingan bozor narxlarini saqlash

**Fields:**

```python
class MarketPriceReference(models.Model):
    # Primary parametrlar
    etaj = IntegerField()  # Qavat raqami (1, 2, 3, ...)
    xonalar_soni = IntegerField()  # Xonalar soni (1, 2, 3, 4)
    qurilish_turi = CharField(max_length=50, choices=[
        ('gishtli', 'Gʻishtli'),
        ('panelli', 'Panelli'),
        ('monolitli', 'Monolitli'),
        ('blokli', 'Blokli'),
    ])
    holat = CharField(max_length=20, choices=[
        ('remontli', 'Remontli'),
        ('remontsiz', 'Remontsiz'),
    ])
    
    # Maydon diapazon
    maydon_min = IntegerField()  # maximal maydon (m²)
    maydon_max = IntegerField(null=True)  # Maksimal maydon (m²)
    
    # Narxlar (so'm/m²)
    arzon_narx = DecimalField(max_digits=12, decimal_places=2)
    bozor_narx = DecimalField(max_digits=12, decimal_places=2)
    qimmat_narx = DecimalField(max_digits=12, decimal_places=2)
    
    # Metadata
    source_file = CharField(max_length=200, blank=True)  # Qaysi fayldan
    created_at = DateTimeField(auto_now_add=True)
    updated_at = DateTimeField(auto_now=True)
    
    # Meta options
    unique_together = ['etaj', 'xonalar_soni', 'qurilish_turi', 'holat', 'maydon_min']
    indexes = [
        Index(fields=['qurilish_turi', 'xonalar_soni']),
        Index(fields=['etaj', 'holat']),
    ]
```

**Methods:**
- `get_narx_range()` → dict: {'min': float, 'avg': float, 'max': float}
- `__str__()` → string: "Gʻishtli - 2 xona - 1-etaj - Remontli"

### Model 2: PropertyPriceAnalysis

**Maqsad:** Mulk obyektlarining narx tahlil natijalarini saqlash

**Fields:**

```python
class PropertyPriceAnalysis(models.Model):
    # Property reference (sizning CRM Property modelingizga bog'lanadi)
    property_id = IntegerField(db_index=True)
    
    # Tahlil natijalari
    status = CharField(max_length=20, choices=[
        ('juda_arzon', 'Juda Arzon'),
        ('arzon', 'Arzon'),
        ('normal', 'Normal'),
        ('qimmat', 'Qimmat'),
        ('juda_qimmat', 'Juda Qimmat'),
    ])
    
    # Narx taqqoslash
    bozor_narxi = DecimalField(max_digits=12, decimal_places=2)  # Bozor narxi (m²)
    joriy_narxi = DecimalField(max_digits=12, decimal_places=2)  # Hozirgi narx (m²)
    farq_foiz = DecimalField(max_digits=6, decimal_places=2)  # Farq foizda
    farq_summa = DecimalField(max_digits=15, decimal_places=2)  # Farq so'mda
    
    # AI tahlili
    ai_tahlil = TextField(blank=True)  # AI tomonidan yozilgan tahlil
    tavsiya = TextField(blank=True)  # AI tavsiyasi
    confidence_score = DecimalField(max_digits=5, decimal_places=2)  # 0-100
    
    # Reference
    matched_reference = ForeignKey(MarketPriceReference, on_delete=SET_NULL, null=True)
    
    # Metadata
    analyzed_at = DateTimeField(auto_now_add=True)
    
    # Indexes
    indexes = [
        Index(fields=['property_id', '-analyzed_at']),
        Index(fields=['status']),
    ]
```

**Methods:**
- `get_status_color()` → string: Hex color code (masalan: "#00C853")
- `get_recommendation_summary()` → string: Qisqa tavsiya matni
- `__str__()` → string: "Property #123 - Arzon"

```python
    source_name: Fayl nomi

    1. CSV qatorini parse: split(',')
    2. Extract ma'lumotlar:
       - etaj = int(values[1])
       - xonalar = int(values[2])
       - qurilish = values[3]
       - maydon = int(values[4])
       - arzon = Decimal(values[5])
       - bozor = Decimal(values[6])

       - QURILISH_MAP.get(qurilish) -> 'gishtli', 'panelli', etc.
    
    5. maydon_max = maydon_min + 5 ni hisoblash
    
    6. Database'ga saqlash:
       MarketPriceReference.objects.update_or_create(
           etaj=etaj,
           xonalar_soni=xonalar,
           qurilish_turi=qurilish_normalized,
           holat=holat,
           maydon_min=maydon,
           defaults={
               'maydon_max': maydon_max,
               'arzon_narx': arzon,
               'bozor_narx': bozor,
               'qimmat_narx': qimmat,
               'source_file': source_name,
```

```python
        'tavsiya': str,

         * max_tokens: 2000
         * messages: [{"role": "user", "content": prompt}]
    3. Response'ni parse qilish
    4. _parse_analysis() bilan natijani extract qilish
    5. Agar xato -> _simple_analysis() fallback
"""
```

#### 3. `_create_analysis_prompt(self, property_data: dict, market_data: dict) -> str`
```python
"""
Claude uchun prompt yaratish

Prompt struktura:
    - Context: "Siz ko'chmas mulk narxlarini tahlil qiluvchi professional ekspertsiz"
    - Property ma'lumotlari (formatted)
    - Bozor narxlari (formatted)
    - Vazifa:
      1. Narxni taqqoslash
      2. Foiz farqini hisoblash
      3. Status aniqlash
      4. Tavsiya berish
    - JSON response format requirement
    - FAQAT JSON, boshqa matn yo'q

Expected JSON response format:
{
    "status": "arzon",
    "farq_foiz": -15.5,
    "farq_summa": -3500000,
    "tahlil": "...",
    "tavsiya": "...",
    "confidence": 85
}
"""
```

#### 4. `_parse_analysis(self, analysis_text: str, property_data: dict, market_data: dict) -> dict`
```python
"""
Claude response'ni parse qilish

Algorithm:
    1. JSON'ni topish:
       - json_start = analysis_text.find('{')
       - json_end = analysis_text.rfind('}') + 1
    2. JSON extract: json_str = analysis_text[json_start:json_end]
    3. json.loads(json_str)
    4. Result dict yaratish
    5. Agar parse xato -> _simple_analysis() fallback
"""
```

#### 5. `_simple_analysis(self, property_data: dict, market_data: dict) -> dict`
```python
"""
Fallback: Oddiy matematik tahlil (AI ishlamasa)

Algorithm:
    1. joriy_narx = property_data['narx_m2']
    2. bozor_narx = market_data['bozor']
    3. farq_foiz = ((joriy - bozor) / bozor) * 100
    4. farq_summa = (joriy - bozor) * maydon
    5. Status aniqlash:
       - farq < -20% -> 'juda_arzon'
       - farq < -10% -> 'arzon'
       - farq < 10% -> 'normal'
       - farq < 20% -> 'qimmat'
       - farq >= 20% -> 'juda_qimmat'
    6. Tavsiya generatsiya
    7. Return dict
"""
```

### Command 2: analyze_properties.py

**File:** `management/commands/analyze_properties.py`

**Command:** `python manage.py analyze_properties`

**Arguments:**
- `--property-id <ID>`: Bitta property tahlil (optional)
- `--no-ai`: AI ishlatmasdan tahlil (optional, default: AI ishlatiladi)
- `--limit <N>`: N ta property tahlil (optional)

**Logic:**

```python
class Command(BaseCommand):
    help = 'Propertylarni narx bo\'yicha tahlil qilish'
    
    def handle(self, *args, **options):
        """
        Algorithm:
            1. PriceAnalyzer() instance
            2. use_ai = not options['no_ai']
            
            3. Agar property_id berilgan:
               - Property.objects.get(id=property_id)
               - analyzer.analyze_property(property, use_ai)
               - Print natija
            
            4. Aks holda bulk analyze:
               - queryset = Property.objects.all()
               - Agar limit -> queryset[:limit]
               - analyzer.bulk_analyze(queryset, use_ai)
               - Print statistics
        """
```

**Usage:**
```bash
# Bitta property
python manage.py analyze_properties --property-id 123

# Barcha propertylar (AI bilan)
python manage.py analyze_properties

# Birinchi 50 ta (AI'siz)
python manage.py analyze_properties --limit 50 --no-ai
```
